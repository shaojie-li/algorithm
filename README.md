### 二叉树的几种遍历
1. 前序遍历
    > 先访问父节点，再分别访问左右子节点
2. 中序遍历
    > 先访问左子节点，再访问父节点，最后访问右子节点
3. 后序遍历
    > 先分别访问左右子节点，最后访问父节点
4. 层次遍历
    > 顾名思义，就是从上之下，从左到右，一层一层访问
